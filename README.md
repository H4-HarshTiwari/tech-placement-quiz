# Tech Placement Quiz

A 20-question tech placement quiz app covering DSA, Java, Web Dev, DBMS, OS, and CN basics.

## Features
- Sidebar question navigator with attempted/marked/unanswered status
- Mark for review functionality
- Dark/light theme toggle (persisted via localStorage)
- Scored results dashboard with a live Chart.js doughnut chart showing correct/wrong/unattempted breakdown
- Fully responsive layout

## Tech Stack
HTML, CSS, JavaScript, Chart.js
